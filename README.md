📒 Mi Presupuesto

Aplicación web simple para registrar ingresos y gastos personales desde una sola página.

Permite llevar un control básico del dinero, registrar movimientos con fecha y visualizar un resumen financiero de forma rápida e intuitiva.

✨ Funciones

- Registrar ingresos y gastos
- Agregar fecha y descripción
- Categorías predefinidas
- Ver saldo disponible
- Resumen de ingresos y gastos
- Historial de movimientos
- Gastos por categoría
- Guardado automático en el dispositivo

🛠️ Tecnologías

- HTML
- CSS
- JavaScript
- LocalStorage

🚀 Uso

1. Selecciona si es Ingreso o Gasto
2. Ingresa el monto
3. Agrega descripción y fecha
4. Selecciona una categoría
5. Guarda el movimiento

La información queda almacenada automáticamente en el navegador.

📱 Características

- Interfaz simple e intuitiva
- Diseño responsive para móvil y PC
- Aplicación de una sola página
- No requiere cuenta ni conexión a internet
